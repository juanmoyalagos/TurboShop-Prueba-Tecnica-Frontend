# TurboShop Marketplace Frontend - React + TypeScript + Vite

SPA en React + TypeScript (Vite) para explorar repuestos de los diferentes proveedores, con búsqueda, filtros, paginación y actualizaciones en tiempo real vía Server-Sent Events.

## Requisitos
- Node.js 
- Yarn

## Instalación y ejecución
1. Clonar el repositorio:
   ```
   git clone https://github.com/juanmoyalagos/TurboShop-Prueba-Tecnica-Frontend.git
   cd TurboShop-Prueba-Tecnica-Frontend
   ```
2. Instala dependencias:
   ```
   yarn install
   ```
3. Crea el archivo `.env` con la URL del backend:
   ```
   VITE_API_URL=http://localhost:3000
   ```
4. Levanta el proyecto en desarrollo:
   ```
   yarn dev
   ```
