# TurboShop Marketplace Frontend - React + TypeScript + Vite

SPA en React + TypeScript (Vite) para explorar repuestos de los diferentes proveedores, con búsqueda, filtros, paginación y actualizaciones en tiempo real vía Server-Sent Events.

## Requisitos
- Node.js 
- Yarn

## Instalación y ejecución
1. Instala dependencias:
   ```
   yarn install
   ```
2. Crea el archivo `.env` con la URL del backend:
   ```
   VITE_API_URL=http://localhost:3000
   ```
3. Levanta el proyecto en desarrollo:
   ```
   yarn dev
   ```
